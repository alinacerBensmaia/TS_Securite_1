Le fichier de paramètres de tests (ParametresTest.testsettings) contient
les configurations nécessaires à l'exécution de ce test. Avant de démarrer le test, il faut
ajouter ce fichier, présent dans le projet, comme élément de votre solution. 

Voici la procédure :
	- Enregistrer la solution
	- Clique de droite sur la solution
	- Choisir l'option Ajouter->Élément existant
	- Dans le dialogue de sélection, sélectionner le fichier "ParametresTest.testsettings" présent dans le projet. 
	- Le dialogue Paramètres s'ouvrira, il s'agit de cliquer sur Fermer
